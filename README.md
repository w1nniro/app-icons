# Генератор иконок для приложений

Одностраничное веб-приложение (PWA) для разработчиков и дизайнеров. Загрузите одно изображение — получите полный набор иконок стандартных размеров для веб-манифеста, iOS, Android и фавиконок.

Вся обработка выполняется **локально в браузере** через Canvas API. Интернет нужен только при первой загрузке; после установки PWA приложение работает полностью офлайн.

## Возможности

- Загрузка изображений: **PNG**, **JPEG**, **WebP**
- Генерация 12 стандартных размеров: 16, 32, 48, 72, 96, 128, 144, 152, 167, 180, 192, 512 px
- Выбор нужных размеров (чекбоксы)
- **Скругление углов** (0–50%) — для iOS и Android adaptive icons
- **Цвет фона** — подложка под прозрачные изображения
- **Отступ (padding)** — процент полей внутри иконки
- **Вписать / Обрезать** — contain или cover для неквадратных изображений
- Превью всех иконок в сетке в реальном времени
- Скачивание каждой иконки отдельно (`icon-16.png`, `icon-32.png`, …)
- Скачивание всех иконок одним **ZIP-архивом** (`app-icons.zip`)
- **PWA** — можно установить на рабочий стол или домашний экран телефона

## Технологии

- React + Vite
- Tailwind CSS (локально)
- Canvas API (без внешних библиотек обработки изображений)
- [jszip](https://www.npmjs.com/package/jszip) — создание архива
- [file-saver](https://www.npmjs.com/package/file-saver) — скачивание файлов
- [vite-plugin-pwa](https://vite-pwa-org.netlify.app/) — офлайн-режим и установка
- [@fontsource/inter](https://www.npmjs.com/package/@fontsource/inter) — шрифт Inter (без CDN)
- [react-icons](https://react-icons.github.io/react-icons/) — иконки Feather

## Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр production-сборки
npm run preview
```

После `npm run build` статические файлы окажутся в папке `dist/`. Их можно разместить на любом хостинге или открыть локально.

## Установка как PWA

1. Откройте приложение в браузере (Chrome, Edge, Safari).
2. В Chrome/Edge: меню → «Установить приложение» / иконка установки в адресной строке.
3. На iOS (Safari): «Поделиться» → «На экран Домой».
4. На Android (Chrome): «Добавить на главный экран».

После установки приложение кэшируется service worker'ом и работает без интернета.

## Как работает генерация

Функция `generateIcon(sourceImage, size, options)`:

1. Создаёт canvas размером `size × size`
2. Применяет clipping со скруглёнными углами (если задан радиус)
3. Заливает фон (если включён)
4. Вписывает или обрезает изображение с учётом padding
5. Экспортирует результат в PNG через `canvas.toBlob()`

При изменении настроек сетка перегенерируется с debounce 300 мс.
